# Master-Thesis-Temporal-Stability
R-scripts used for the data-analysis in the master thesis Temporal Stability of Macrozoobenthos Biomass in the Baltic Sea Salinity Gradient by Julius Lange.
The repository includes two R-scripts: data_preparation.R and data_exploration.R. 
 data_preparation.R includes the code used to explore and clean up the raw data set. This script must be run first in order to create the finalized dataset before any further analysis can be done.
 data_exploration.R includes all of the code used for the analysis of the cleaned up dataset. The script is divided into topic sections corresponding to different parts of the analysis. Further information is provided in the comments within the code. This script can only be run after the finalized data set has been created using data_preparation.R.
